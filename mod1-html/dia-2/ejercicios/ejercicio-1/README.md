# Ejercicio 1

-   Crea un nuevo repositorio en GitHub que tenga por nombre `ejercicios`.

-   Clona dicho repositorio en la carpeta `powercoders` que deberá estar ubicada
    dentro de `Documents`.

-   Ubícate dentro del repositorio recién clonado y crea un nuevo directorio llamado
    `animales` y otro directorio llamado `colores` (ambos deberán ser hijos del directorio
    `ejercicios`).

-   Dentro de `animales` crea un archivo llamado `favoritos.txt` e introduce dentro tus colores
    animales favoritos. Haz lo mismo en el directorio `colores`.

-   Ahora crea una nueva versión de tu repositorio que incluya todos esos cambios (necesitarás
    hacer un `commit`).

-   Comprueba que el `commit` se ha realizado correctamente mediante el comando correspondiente.

-   Si todo ha ido bien, sube los cambios al respositorio en la nube (a GitHub).

-   Ahora prueba a realizar algún cambio en cualquiera de los dos archivos `favoritos.txt`,
    crea una nueva versión y súbe los cambios a la nube.

-   Elimina el repositorio `ejercicios` tanto en remoto (en GitHub) como en local. ¿Cómo se elimina
    un repositorio en GitHub? Ah, toca buscar en Google... ;)
