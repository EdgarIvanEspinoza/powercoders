## Según

<img src="https://firebasestorage.googleapis.com/v0/b/virtually-1f5e0.appspot.com/o/dashboard%2Fprofile%2F9002ejercicio-2.png?alt=media&token=1af9f51b-d604-4f5d-94ab-8ed9e1513191" width=1000 />

- Iniciamos el algoritmo.

- Asignamos a miValor un 1.

- Analizamos el contenido de miValor:

  - Si miValor es igual a 1 se muestra el valor "Perro".

  - Si miValor es igual a 2 se muestra el valor "Gato".

  - Si miValor es igual a 3 se muestra el valor "Ratón".

  - Si miValor no es igual a alguna de las opciones anteriores se muestra un mensaje indicando que no se reconoce el valor introducido.

- Finaliza el algoritmo.

## Para

- Iniciamos el algoritmo.

- En el óvalo fucsia escribimos el nombre de la variable o contador, en este caso miValor. También definimos el valor inicial, el incremento en el contador tras cada iteración, y el valor final para el contador.
